# 🎤 AI Interview Assistant

## 📌 Project Description
This is a web-based AI Interview Assistant built using Python and Streamlit.  
It asks interview questions, collects answers, and provides feedback to help users improve their interview skills.

---

## 🚀 Features
- Ask interview questions one by one
- Accept user answers
- Provide feedback after each answer
- Show overall performance at the end
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- Python
- Streamlit

---

## ▶️ How to Run
1. Install Python
2. Install Streamlit:
   pip install streamlit
3. Run the project:
   streamlit run app.py

---

## 📊 Output
- Displays questions
- Takes user input
- Gives feedback
- Shows final result
